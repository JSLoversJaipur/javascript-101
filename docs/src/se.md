# linux


```
Visual Studio Code can be installed on various Linux distributions using different package managers. Here's how to install it using a few common package managers:
```
```
1.Debian/Ubuntu:
Open a terminal and run the following commands:
```
```
sudo apt update 

sudo apt install code

```
```
2.Fedora:
Open a terminal and run the following command:
```
```
sudo dnf install code
```
```
3.Arch Linux:
Open a terminal and run the following command:
```
```
sudo pacman -S code
```
