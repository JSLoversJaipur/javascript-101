# Macos setup



```
1. Visit the official Visual Studio Code website: https://code.visualstudio.com/

2. Click on the "Download for macOS" button. This will download a .zip file containing the VSCode app.

3. Locate the downloaded file (usually in your "Downloads" folder) and double-click on it to extract the app.

4. Drag the VSCode app to the Applications folder to install it.

5. After installation, you can find VSCode in your Applications folder. You can also add it to the Dock for quick access.
```
